# Habitly

**Habitly** is a modern habit tracking web application designed to help users build consistency, improve productivity, and maintain positive daily routines. It provides a clean and motivating environment where users can track habits, monitor streaks, and visualize their progress over time.

The application focuses on **simplicity, clarity, and motivation**, making it easier for users to stay committed to their goals and develop long-term habits.

---

## Key Features

### Habit Tracking
Create and manage daily habits easily. Habitly allows users to define personal goals and track whether they complete them each day through a simple and intuitive interface.

### Daily Progress Monitoring
Users can mark habits as completed with a single click. The system records completion data for each day, allowing users to clearly see their activity and consistency.

### Streak System
Habitly includes a motivational streak system that helps reinforce consistency.

- **Current Streak** – Number of consecutive days a habit has been completed.
- **Best Streak** – The longest streak achieved for that habit.

These streaks provide immediate feedback and encourage users to maintain their routines.

### Weekly Habit Tracker
The weekly tracker displays habits in a grid format where each day of the week can be marked as completed. This provides a quick overview of the user’s weekly consistency.

### Yearly Progress Visualization
Habitly provides a **GitHub-style yearly heatmap** that visualizes long-term habit activity. Each day is represented with color intensity based on completion, allowing users to identify patterns and track progress over months.

### Visual Feedback
Color-coded progress indicators and streak counters help users instantly understand their habit performance and motivate them to maintain consistency.

### Clean and Minimal Interface
Habitly is designed with a minimal and distraction-free interface that prioritizes clarity and ease of use.

### Responsive Experience
The application works smoothly across devices including desktops, tablets, and smartphones, ensuring users can track habits anytime.

---

## Why Habitly?

Building habits requires consistency and visibility. Habitly helps users:

- Stay accountable to their daily routines
- Visualize progress over time
- Maintain streaks that reinforce positive behavior
- Develop long-term productive habits

The goal of Habitly is to make habit tracking **simple, motivating, and visually engaging**, helping users gradually improve their daily lives.
